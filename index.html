This variable should be an object with three properties: a string (name), a number (age), and a function (greeting). The function should return, "Hello, my name is [name] and I am [age] years old." Outside of the immediately invoked function, set the name and age properties of Introduction, then log the result of the greeting method to the console. 2. Depending on the user's answer of yes or no, the automated bot should be able to produce a horoscope or say goodbye. The horoscope can be as simple or complex as you'd like. Please use CSS (do not use Bootstrap or any similar framework) to style the conversation to look like a traditional chat based interface (iMessage, for example).
